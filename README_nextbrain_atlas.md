# NextBrain Whole-Brain Atlas in MNI152 Space

## Files

| File | Description |
|------|-------------|
| `nextbrain_wholebrain_MNI152_1mm.nii.gz` | Whole-brain parcellation in MNI152 1mm space (496 ROIs) |
| `nextbrain_wholebrain_MNI152_1mm_lut.txt` | Lookup table mapping label indices to region names |

---

## Overview

This atlas was generated by running the [NextBrain](https://github-pages.ucl.ac.uk/NextBrain) segmentation tool on the MNI152 T1 1mm template. NextBrain is a probabilistic histological atlas of the human brain built from ~10,000 histological sections of 5 complete brain hemispheres, providing up to 333 ROIs per hemisphere (Casamitjana et al., *Nature*, 2025).

The two hemispheric segmentations were resampled to MNI152 1mm space and merged into a single parcellation volume. Right hemisphere labels are offset by 10,000 to distinguish them from left hemisphere labels (e.g., label `48` = left head of caudate, label `10048` = right head of caudate).

## Generation Pipeline

### Requirements

- [FreeSurfer 8.1.0](https://surfer.nmr.mgh.harvard.edu/) (Rocky Linux 9 tarball)
- [Apptainer](https://apptainer.org/) (or Singularity)
- FreeSurfer license file (`license.txt`)
- NextBrain atlas files: [atlas_simplified.zip](https://ftp.nmr.mgh.harvard.edu/pub/dist/lcnpublic/dist/Histo_Atlas_Iglesias_2023/atlas_simplified.zip)
- MNI152 T1 1mm template (available in FSL: `$FSLDIR/data/standard/MNI152_T1_1mm.nii.gz`)

### Step 1 — Build FreeSurfer Apptainer image

```bash
apptainer build freesurfer_8.1.0.sif freesurfer_8.1.0.def
```

The `.def` file installs FreeSurfer 8.1.0 from the official Rocky Linux 9 tarball:

```singularity
Bootstrap: docker
From: rockylinux:9

%environment
    export FREESURFER_HOME=/opt/freesurfer
    export PATH=$FREESURFER_HOME/bin:$PATH

%post
    dnf install -y --allowerasing curl tcsh bc binutils libgomp \
        perl python3 mesa-libGLU libXrender libXt libXmu && dnf clean all
    curl -o /tmp/freesurfer.tar.gz \
        https://surfer.nmr.mgh.harvard.edu/pub/dist/freesurfer/8.1.0/freesurfer-linux-rocky9_x86_64-8.1.0.tar.gz
    tar -xzf /tmp/freesurfer.tar.gz -C /opt/
    rm /tmp/freesurfer.tar.gz
```

### Step 2 — Download NextBrain atlas files

```bash
wget https://ftp.nmr.mgh.harvard.edu/pub/dist/lcnpublic/dist/Histo_Atlas_Iglesias_2023/atlas_simplified.zip
unzip atlas_simplified.zip
```

### Step 3 — Run segmentation on MNI152 template

Submitted as a SLURM job on a node with NVIDIA A100 SXM4 (80GB VRAM):

```bash
apptainer exec \
  --nv \
  --bind /path/to/license.txt:/opt/freesurfer/license.txt \
  --bind /path/to/atlas_simplified:/opt/freesurfer/python/packages/ERC_bayesian_segmentation/atlas_simplified \
  freesurfer_8.1.0.sif \
  mri_histo_atlas_segment_fireants \
    MNI152_T1_1mm.nii.gz \
    output/ \
    1 \
    16
```

This produces `seg.left.nii.gz`, `seg.right.nii.gz`, and `lut.txt` in the output directory.

### Step 4 — Merge hemispheres into a single MNI152 volume

```python
import nibabel as nib
import numpy as np
from nibabel.processing import resample_from_to

mni_nii = nib.load('MNI152_T1_1mm.nii.gz')
left_nii = nib.load('output/seg.left.nii.gz')
right_nii = nib.load('output/seg.right.nii.gz')

# Resample both hemispheres to MNI152 space (nearest-neighbour for labels)
left_mni = resample_from_to(left_nii, mni_nii, order=0)
right_mni = resample_from_to(right_nii, mni_nii, order=0)

left = left_mni.get_fdata().astype(int)
right = right_mni.get_fdata().astype(int)

# Right hemisphere labels offset by 10000
OFFSET = 10000
merged = np.zeros_like(left)
merged[right > 0] = right[right > 0] + OFFSET
merged[left > 0] = left[left > 0]  # left takes priority in overlap

out = nib.Nifti1Image(merged.astype(np.int32), mni_nii.affine, mni_nii.header)
nib.save(out, 'nextbrain_wholebrain_MNI152_1mm.nii.gz')
```

## Label Convention

- Labels `1–9999` → **left hemisphere** ROIs (original NextBrain indices)
- Labels `10001–19999` → **right hemisphere** ROIs (original index + 10000)
- Label `0` → background

The `lut.txt` file uses the format:
```
index  region_name  R  G  B  A
```
